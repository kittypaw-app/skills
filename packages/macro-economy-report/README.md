# Macro Economy Report

Sends a daily macro-economic briefing to a Telegram chat. Fetches real-time price data for major ETFs (equities, bonds, gold) via the Yahoo Finance API, then uses an LLM to write a concise market commentary.

## What it does

1. Fetches closing prices and daily change for each configured ticker (default: SPY, QQQ, TLT, GLD).
2. Passes the snapshot to the LLM to generate a 3–5 sentence neutral market commentary.
3. Sends the formatted report to Telegram every weekday morning at 08:00.

## Configuration

| Key | Type | Required | Default | Description |
|---|---|---|---|---|
| `telegram_token` | secret | yes | — | Your Telegram bot token |
| `chat_id` | string | yes | — | Telegram chat or channel ID |
| `tickers` | string | no | `SPY,QQQ,TLT,GLD` | Comma-separated Yahoo Finance ticker symbols |
| `schedule` | cron | no | `0 8 * * 1-5` | When to run (weekdays at 08:00 by default) |

## Data Source

- **Yahoo Finance v8 Chart API** — `https://query1.finance.yahoo.com/v8/finance/chart/{ticker}`
- Free, no API key required.
- LLM commentary generated by the built-in `Llm.generate` primitive.
