# 오늘의 뉴스 요약 (news-digest-kr)

매일 아침 8시, 구글 뉴스 코리아 RSS에서 선택한 카테고리의 주요 기사를 가져와 LLM으로 각 헤드라인을 한 줄 요약한 뒤 텔레그램으로 전송합니다.

## 동작 방식

1. Google 뉴스 Korea RSS에서 설정한 카테고리(정치/경제/사회/IT과학/세계)의 최신 기사를 가져옵니다.
2. 설정한 `count`만큼 상위 기사 제목을 추출합니다.
3. LLM에게 각 제목을 한 줄로 요약하도록 요청합니다.
4. 번호 목록 형식으로 정리해 텔레그램 메시지로 발송합니다.

## 설정

| Key | Type | Required | Default | Description |
|---|---|---|---|---|
| `telegram_token` | secret | yes | — | 텔레그램 봇 토큰 |
| `chat_id` | string | yes | — | 텔레그램 채팅 또는 채널 ID |
| `category` | select | no | `경제` | 뉴스 카테고리 (정치/경제/사회/IT과학/세계) |
| `count` | number | no | `5` | 요약할 기사 수 |

## 출력 예시

```
📰 오늘의 뉴스 5선 (경제)
2026년 3월 31일

1. 한국은행, 기준금리 동결 결정 — 경기 불확실성 지속 반영
2. 삼성전자 1분기 영업이익 컨센서스 상회 전망
3. 코스피 2,600선 회복 시도, 외국인 순매수 전환
4. 원/달러 환율 1,350원대 안착, 수출 기업 수혜 기대
5. 정부, 반도체 특별법 연내 처리 추진 방침 발표

출처: Google 뉴스 · Powered by KittyPaw
```
