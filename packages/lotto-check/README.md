# 로또 당첨 확인

매주 토요일 밤 9시에 로또 6/45 최신 회차 당첨 번호를 자동으로 조회하고, 내 번호와 비교한 결과를 텔레그램으로 알려줍니다.

## 동작 방식

1. 매주 토요일 21:00에 동행복권 API에서 최신 회차 당첨 번호를 가져옵니다.
2. 설정한 내 번호 6개와 당첨 번호를 비교합니다.
3. 일치 개수와 보너스 번호 여부로 등수를 판정합니다.
4. 결과를 텔레그램으로 발송합니다.

## 설정

| Key | Type | Required | Description |
|---|---|---|---|
| `telegram_token` | secret | yes | 텔레그램 봇 토큰 |
| `chat_id` | string | yes | 텔레그램 채팅 또는 채널 ID |
| `my_numbers` | string | yes | 쉼표로 구분된 6개 번호 (예: `3,8,15,23,29,42`) |

## 당첨 기준

| 등수 | 조건 |
|---|---|
| 1등 | 6개 모두 일치 |
| 2등 | 5개 일치 + 보너스 번호 일치 |
| 3등 | 5개 일치 |
| 4등 | 4개 일치 |
| 5등 | 3개 일치 |
| 낙첨 | 2개 이하 일치 |

## 알림 예시

```
🎰 로또 1162회 당첨 번호
3, 8, 15, 23, 29, 42 + 보너스 7

내 번호: 3, 8, 15, 23, 29, 42
일치: 6개 — 🏆 1등!
```
