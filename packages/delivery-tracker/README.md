# 배송 조회

택배 배송 상태를 2시간마다 자동으로 확인하고, 상태가 바뀔 때만 텔레그램으로 알려줍니다. 스마트택배 API를 사용하며 CJ대한통운, 한진택배, 롯데택배, 우체국 4개 택배사를 지원합니다.

## 동작 방식

1. 2시간마다 스마트택배 API로 운송장 번호의 최신 배송 상태를 조회합니다.
2. 이전 상태를 Storage에 저장해두고, 새 상태와 비교합니다.
3. 상태가 바뀐 경우에만 텔레그램 메시지를 발송합니다.
4. 변경이 없으면 조용히 종료합니다.

## 설정

| Key | Type | Required | Description |
|---|---|---|---|
| `telegram_token` | secret | yes | 텔레그램 봇 토큰 |
| `chat_id` | string | yes | 텔레그램 채팅 또는 채널 ID |
| `carrier` | select | yes | 택배사 (CJ대한통운 / 한진택배 / 롯데택배 / 우체국) |
| `tracking_number` | string | yes | 운송장 번호 |
| `api_key` | secret | yes | 스마트택배 API 키 |

## 스마트택배 API 키 발급

1. [스마트택배](https://info.sweettracker.co.kr) 에 가입합니다.
2. 개발자 센터에서 API 키를 발급받습니다.
3. 발급받은 키를 `api_key` 설정 항목에 입력합니다.

## 알림 예시

```
📦 배송 조회 — CJ대한통운
1234567890123

현재: 배송출발
2025-03-28 08:42
```
